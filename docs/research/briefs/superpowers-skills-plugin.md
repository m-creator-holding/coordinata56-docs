## Что это делает

Готовый набор навыков («скилов») для Claude Code, которые автоматически включаются в нужный момент работы: обдумывание задачи, разработка через тесты (TDD), систематическая отладка, управление параллельными веточками git (worktree), планирование, код-ревью, координация нескольких агентов. Аналогия: как «служебка» с разделами «что делать, если сломалось», «как браться за новую задачу», «как проверять работу другого» — только эта служебка сама подсовывается агенту в подходящий момент, без ручного включения. Ставится одной командой `/plugin install superpowers@claude-plugins-official`.

## Чем это полезно coordinata56

У нас уже есть требование «reviewer до `git commit`» и фазовая работа по DoD — Superpowers закрывает ту же задачу готовым набором процедур. Потенциально ускоряет onboarding новых субагентов и снимает часть ручного контроля с Координатора: TDD-скилы напоминают написать тест прежде кода, debugging-скил — собрать контекст ошибки перед правкой. Особенно интересно для backend-dev и qa.
