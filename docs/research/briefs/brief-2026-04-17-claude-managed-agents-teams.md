## Что это делает

Anthropic в апреле 2026 года запустила новый сервис «Управляемые агенты» (Managed Agents) с функцией «Команды агентов» (Agent Teams). Это как нанять несколько независимых бригад вместо одной: у каждой своя память, свой список задач, и они могут напрямую общаться друг с другом — без прораба-посредника. Цена: $0.08 за час работы плюс обычная оплата за токены (слова, которые обрабатывает ИИ). Доступно в публичной бете через специальный заголовок API.

## Чем полезно нам

Это потенциальное решение для главного ограничения нашей системы: сейчас Директор не может напрямую дать задачу Worker'у без участия Координатора. Agent Teams позволяет создать архитектуру, где агенты одного уровня общаются напрямую. До принятия решения нужен анализ: стоимость, безопасность, совместимость с нашим Claude Code (не облачным API).

## Что предлагаем

1. Передать ri-analyst для оценки: совместим ли Agent Teams с нашей on-premise моделью (Claude Code в терминале) или это только для API-интеграций.
2. Если совместим — рассмотреть как альтернативу паттерну «Координатор-транспорт» для горизонтальных коммуникаций Директор-Директор.
