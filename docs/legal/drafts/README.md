# Юридические черновики для Production Gate M-OS

Все документы этой папки — **черновики**, подготовленные AI-помощником на основе legal-checklist'а (`docs/legal/m-os-1-1-foundation-legal-check.md`). Каждый черновик начинается с явного дисклеймера об этом. Документы передаются живому юристу для проверки, доработки и утверждения.

## Зависимости между документами

```
D-1 (Политика конфиденц.) ──────────────┐
                                         ├──► D-3 (Согласие работника)
D-2 (Положение об обраб. ПД) ───────────┘
         │
         └──► D-4 (Уведомление РКН) ──── подаётся после утверждения D-1, D-2

D-5 (Политика мониторинга) ─────────────► D-6 (Уведомление работника)
```

Логический порядок утверждения живым юристом: D-2 → D-1 → D-3 → D-5 → D-6 → D-4.  
D-4 (Уведомление РКН) подаётся последним — после того как все внутренние документы утверждены.

## Инструкция для живого юриста

Уважаемый юрист,

Вам передаётся пакет из 6 черновиков юридических документов, необходимых для открытия production gate корпоративной информационной системы M-OS. Черновики подготовлены AI-помощником по структуре и содержанию, обязательные разделы включены, однако каждый документ требует Вашей профессиональной проверки.

**Что нужно сделать:**

1. Прочитать каждый черновик и разделы «Открытые вопросы для юриста» в каждом из них.
2. Заполнить плейсхолдеры (список выше) совместно с Владельцем.
3. Исправить формулировки, не соответствующие актуальной редакции закона.
4. Оценить, нужны ли дополнительные документы (например, D-7 — Соглашение об использовании НЭП, D-8 — Регламент реагирования на утечку ПД; они не включены в данный пакет).
5. Утвердить каждый документ подписью.
6. Организовать подачу Уведомления в Роскомнадзор (D-4) через портал pd.rkn.gov.ru.

**Критичные вопросы, требующие Вашего однозначного ответа:**

- Единый оператор или отдельный оператор на каждое юрлицо холдинга?
- Является ли использование Telegram-бота трансграничной передачей ПД по смыслу ст. 12 152-ФЗ?
- Достаточно ли трудового договора как основания для обработки данных аудит-журнала M-OS, или нужно отдельное согласие?
- Уровень защищённости ИСПДн по ПП № 1119?
