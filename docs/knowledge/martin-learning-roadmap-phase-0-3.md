## Часть 5. Глоссарий новых терминов

| Термин | Простое объяснение |
|--------|--------------------|
| **Терминал** | Окно для текстовых команд компьютеру. Аналог рации с прорабом. |
| **Git** | Система контроля версий. Журнал всех изменений кода с датой и подписью. |
| **Коммит** | Зафиксированный «снимок» состояния кода с описанием изменения. |
| **Docker** | Инструмент для запуска программ в изолированных контейнерах. |
| **Контейнер** | Изолированная «бытовка» для одной программы со своей средой. |
| **HTTP** | Протокол передачи данных в интернете. |
| **API** | Набор «окошек» сервера для запросов данных или команд. |
| **Эндпоинт** | Одно конкретное «окошко» API, например `POST /api/auth/login`. |
| **JSON** | Формат данных: словарь с ключами и значениями. |
| **JWT** | Токен-бейдж: зашифрованная строка с данными о пользователе и подписью сервера. |
| **Фреймворк** | Готовый скелет программы. Добавляете только специфическую логику. |
| **async/await** | Асинхронность: программа не замирает в ожидании, делает другие задачи. |
| **SQL** | Язык запросов к реляционной БД. SELECT, WHERE, JOIN. |
| **Swagger UI** | Интерактивная документация FastAPI. Доступна по `/docs`. |
| **Middleware** | Промежуточный слой в обработке запроса (например, проверка токена). |
