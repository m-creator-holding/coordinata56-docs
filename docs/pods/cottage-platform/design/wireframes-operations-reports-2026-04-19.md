## Принятые решения

| # | Точка выбора | Выбрано | Обоснование |
|---|---|---|---|
| R1 | Доступ foreman к /reports/financial | Серверный редирект на /dashboard | Не показываем страницу вообще. Навигационный пункт «Отчёты» скрыт в sidebar для foreman. |
| R2 | Drill-down по категории | Sheet справа (shadcn/ui) | Не отдельная страница: контекст сводной таблицы остаётся виден слева. Sheet закрывается кнопкой ×. |

| R4 | Фильтр детализации | Select: «По месяцу / По кварталу» | Два уровня покрывают 90% запросов. Не усложняем на skeleton. |
| R5 | «ИТОГО» строка | Sticky bottom в DataTable | Всегда видна при прокрутке категорий. Реализация: CSS sticky + bold. |

## Навигационная карта

```
/reports/financial              ← этот файл
  ├── клик на строку категории  → Sheet с drill-down (inline, не переход)
  ├── [Экспорт] → CSV           → скачивание (enabled)
  ├── [Экспорт] → Excel         → скачивание (enabled)
  └── [Экспорт] → 1С            → disabled, tooltip

foreman → редирект на /projects/{projectId}/overview
```

## Экран 3. Financial Reports — `/reports/financial`

### Persona и сценарий

**Персона — Бухгалтер (accountant):**
Открывает отчёт, выбирает период «Январь–Апрель 2026», детализация «По месяцу». Видит сводную таблицу: план/факт/дельта по категориям. Кликает на «Стройматериалы» — Sheet справа показывает разбивку по домам. Экспортирует в Excel для руководителя.

**Персона — Владелец/Директор:** тот же сценарий + возможность видеть все категории без ограничений.

### API-эндпоинты

```
GET /api/v1/reports/financial
  ?project_id={id}&date_from=2026-01-01&date_to=2026-04-30&group_by=month
  → { categories: [{name, plan, fact, delta, pct}],
      total: {plan, fact, delta, pct} }

GET /api/v1/reports/financial/category/{categoryId}/drilldown
  ?project_id={id}&date_from=...&date_to=...
  → { category: "materials", items: [{house_id, house_number, plan, fact}] }

GET /api/v1/reports/financial/export?format=csv|excel&...
  → file download (Content-Disposition: attachment)
```
