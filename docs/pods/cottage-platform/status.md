## Что закрыто сегодня (2026-04-19)

### Sprint 1 (US-01, US-02, US-03)
Фундамент многокомпанийной архитектуры. Данные каждой компании изолированы на уровне базы данных. Права управляются через матрицу (не в коде). JWT-токен несёт список компаний пользователя.

### Wave 11 + Round 3/4 регрессия
Административная панель: 4 страницы (матрица прав, компании, пользователи, бизнес-правила). Четыре бага закрыты (авторизация Владельца, фильтрация, миграция, IDOR-уязвимость). Hardcoded-пароли из тестов убраны.

### WAL Fix B
Тройная защита данных:
- WAL-архив в Yandex Object Storage — непрерывно, каждый блок транзакций
- SQL-дамп — ежедневно, хранение 30 дней
- Том Docker — локально

### Sprint 2 Volna A (US-04, US-05, US-06, US-07)
Событийный слой и интеграционный каркас:
- `BusinessEventBus` — деловые события (платёж, договор, стадия дома)
- `AgentControlBus` — управление ИИ-субагентами
- ACL IntegrationAdapter — защита от чужих форматов на границе системы
- Pluggability container — подключение новых модулей без правки ядра

### Дополнительно
- ADR-0024 Verification Gate: правила доказательства live-активаций
- ADR-0025: архитектура интеграции с 1С (черновик)
- 1С adapter skeleton: 60 тестов на полке, живых вызовов нет
- 3 вайрфрейма Operations UI: дашборд / дома / отчёты
- Email-маска `mask_email` добавлена к защите персональных данных
