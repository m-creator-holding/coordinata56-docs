## 2. Ключевые решения периода

### Решение 1: ADR-0014 ratified — backup-mode

**Статус:** accepted (force-majeure, governance-auditor backup-approver)  
**Механизм:** governance-director dormant в 5-й раз за 2 дня → governance-auditor выступил backup-approver согласно регламенту.  
**Следствие:** Gate-0 разблокирован. Код M-OS-1.1A может стартовать.  
**Ретроспективный approve:** при восстановлении governance-director.

### Решение 2: Top-3 паттернов Odoo для заимствования

Вердикт innovation-director: **REJECT как альтернативу + MIX (заимствовать паттерны)**.

1. **Developer mode** — runtime-кастомизация полей через UI, без деплоя → паттерн для Admin UI конструктора M-OS.
2. **BoQ + Work Packages + Cost Center** — структура стройпроекта → референс для `Project → Phase → WorkPackage → Task` в cottage-pod.
3. **Kanban + notebook + calendar** — три базовых представления для каждой сущности → UX-паттерн для frontend.

### Решение 3: Yandex SpeechKit финалист Voice AI

**Рекомендация innovation-director:** Yandex SpeechKit (потоковое v3 + дообучение на строительном глоссарии).  
**Резерв:** Sber SaluteSpeech.  
**Офлайн-fallback:** Whisper (open-source) — только при отсутствии сети, не основной канал.  
**Оценка стоимости:** 60–90 тыс. руб/мес при 85 домах × 50 записей × 1 мин.  
**Статус:** ожидает решения Владельца. В план M-OS-2 (Trial T1 Technology Radar).
